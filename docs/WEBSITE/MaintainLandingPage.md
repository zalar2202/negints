## 3. What We Do (Service Capabilities)
- **Security Patching**: WordPress core/plugins, library updates (npm), OS patches.

- **Performance Optimization**: Regular database cleanup, image optimization.
- **Bug Fixes**: Priority resolution for any issues that arise.
- **Content Updates**: Assistance with posting blogs, changing images, etc.

## 4. Support Process (The Cycle)
1.  **Monitor**: Continuous automated scanning.
2.  **Detect**: Identifying anomalies or outdated packages.
3.  **Resolve**: Fixing issues before they impact users.
4.  **Report**: transparent monthly executive summaries.
5.  **Optimize**: Proactive suggestions for improvement.

## 5. Deliverables
- **Monthly Health Reports**: PDF summary of uptime, traffic, and work done.

- **Priority Support Access**: Direct line to engineers (Slack/Phone).
