## üíª Technical Implementation

The website (`negints-app`) is built with a modern Next.js stack:

- **Framework:** Next.js 14+ (App Router)
- **Database:** MongoDB (Mongoose)
- **UI Architecture:** Responsive custom CSS with Tailwind support where necessary.
- **Key Sections:**
    - `Hero` - Medical theme landing with particle effects.
    - `Products List` - Dynamic filtering by Category ID and legacy fields.
    - `Product Details` - Rich technical specs and related products.
    - `FAQAccordion` - Standard inquiries for healthcare clients.
    - `GlobalCTA` - Unified contact point for medical consultations.
